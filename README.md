# Student Manager System

## Giới thiệu
Ứng dụng quản lí sinh viên với hệ thống indexing tối ưu hiệu năng, được phát triển cho môn cấu trúc dữ liệu và giải thuật.

## Tính năng 
- Quản lí sinh viên (CRUD)
- Indexing tối ưu tìm kiếm
- Xét học bổng tự động
- Giao diện console tiếng Việt

## Công nghệ
- Java 8+
- Hashmap + Indexing
- Collections Framework
